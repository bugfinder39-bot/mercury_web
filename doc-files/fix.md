Perform a complete visual redesign of the entire website and admin panel while maintaining the existing functionality. The goal is to create a premium, modern, and professional corporate UI using a consistent **Dark Navy + Orange** brand identity.

## Overall Theme

- Dark Navy should be the primary brand color.
- Orange should be the primary accent color.
- White and light gray should be used only for content backgrounds.
- Every page should have a premium corporate appearance.
- Maintain consistency across the entire project.

## Icons

Refactor every icon used throughout the project.

Requirements:

- Change every icon color to the brand Orange (#E8770C).
- Use one consistent icon library.
- Icons should immediately attract attention without overwhelming the UI.
- Icons inside buttons, cards, service sections, contact information, statistics, timeline, dashboard, forms, footer, navigation, CMS, and admin pages should all follow this rule.
- Icons should have consistent sizing and spacing.

## Shadows

Every component should have subtle depth.

Requirements:

- Add soft edge shadows.
- Use professional low-opacity shadows.
- Components should visually "pop" from the background.
- Avoid heavy floating shadows.
- Maintain a premium enterprise look.

Example style:

- shadow-sm
- shadow-md
- custom shadow with low opacity

Hover state may slightly increase the shadow.
