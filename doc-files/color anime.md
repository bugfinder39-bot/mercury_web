==================================================
DESIGN GOAL
==================================================

Create a modern, colorful, premium corporate website for a shipping and logistics company.

The client wants the website to feel more vibrant and visually engaging while remaining professional, clean, and trustworthy.

The website should look like it was designed by a high-end digital agency.

It should immediately communicate:

• Professional
• Modern
• Reliable
• Premium
• Dynamic
• International

MAKE THE WEBSITE MORE COLORFUL
==================================================

The website currently feels flat and mostly white.

Increase visual impact by using:

• Alternating white and navy sections
• Orange accent backgrounds
• Soft navy gradients
• Orange highlight elements
• Colored feature cards
• Decorative background shapes
• Abstract wave dividers
• Geometric patterns
• Subtle SVG graphics
• Accent lines
• Colored badges
• Section highlights
• Image overlays

==================================================
GLASSMORPHISM
==================================================

Use subtle glass effects only where appropriate.

Examples:

• Sticky Navbar
• Hero Content Card
• Floating Statistics
• CTA Cards
• Dashboard Summary Cards

Glass effect should include:

• Backdrop blur
• Semi-transparent white background
• Thin white border
• Soft shadow

Do NOT apply glass effects everywhere.

==================================================
SHADOWS
==================================================

The website needs more depth.

Add elegant shadows to:

• Service Cards
• Feature Cards
• Gallery Cards
• CEO Section
• Testimonials
• Contact Form
• Buttons
• Statistics Cards
• Partner Cards
• Dashboard Widgets

Hover states should slightly increase elevation.

Use soft premium shadows.
